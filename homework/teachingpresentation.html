The difference between Head, Header, and Heading tags

1. Head

  • The HTML head element has nothing to do with HTML headings.

  • Head tag comes before the header tag and goes before body tag.

  • Head tag contains information about web page.

  • We don’t put any content inside the head tag.

2. Header

The header element represents a container for introductory content or a set of navigational links. Header tag also is used as container for html heading. You can have several header elements in one document.

A header element typically contains:

  • One or more heading elements (h1 - h6)
  • Logo or icon
  • Authorship information

3. Heading

There are six html heading tags.h1 defines the most important heading. h6 defines the least important heading.In other words it is smaller. By using CSS, we can make the smaller headings bigger.

  • h1 defines the most important heading.
  • headings should be used for main headings.
  • defines the least important heading.

Why html headings are Important?

  • It shows the document structure and contents.
  • Search engines use the headings to index the structure and content of your web pages.
  • Users skim your pages by its headings.

Documents outlines

The document outline is the structure of a document, generated by the document’s headings, form titles, table titles, and any other appropriate landmarks to map out the document.Documents outline also shows how the contents and information are organized.
